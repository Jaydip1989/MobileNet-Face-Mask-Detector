# MobileNet-Face-Mask-Detector

For this project I have used the kaggle dataset of New Face Mask and I have used the pretrained MobileNet weights for this project.

I have also haarcascade frontalface file for dtecting the faces.

In given an image I recieved and accuracy of 97% .
